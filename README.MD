# [iTrace](https://github.com/i5wear/iTrace)

适用于多种场合的要塞计算器，由 i5wear 制作。

基于准确的数学模型，其结果置信度不低于任何同类产品。

本产品的研发和制作基于世界生成算法库 [Cubiomes](https://github.com/Cubitect/Cubiomes)。

#### 使用帮助

程序首次运行时，请先设定游戏版本，并设定误差或开始误差校准。

对准末影之眼按 F3 + C，直接粘贴至本程序，即可快捷输入数据。

若上述方式无法使用，则按下 F3，手动输入坐标和角度至本程序。

正常创建一个世界，获取种子并粘贴至本程序，即可开始误差校准。

终止本程序不会丢失任何数据，所有历史输入均被自动保存与读取。

#### 命令列表

添加末影之眼：ADD [X] [Z] [Yaw]

设定游戏版本：VER 1.[N]

设定测量误差：ERR [Mean] [SD]

开始误差校准：CAL [Seed]

终止误差校准：CAL 0

查看所有数据：CHECK

清除所有数据：CLEAR

终止程序运行：空输入
